## 📖 Project Overview

A modern and seamless digital library web application designed to transform the traditional library experience into an interactive online platform. Users can explore a wide collection of books, filter by categories, and borrow books digitally.

This project focuses on performance, security, and modern UI/UX using cutting-edge technologies.

---

## 🎯 Purpose

- Digitize traditional library systems
- Provide an easy way to browse and borrow books
- Build a scalable and responsive full-stack application

---

## 🚀 Key Features

### 🔐 Authentication

- Secure user authentication system
- Protected routes (dashboard)

### 📚 Book Management

- View all books
- View single book details
- Add & remove books (API based)

### 🔍 Search & Filter

- Search books by title or author
- Filter by category

### 🔄 Borrow System

- Borrow books with one click
- Availability status update

### 📱 Responsive Design

- Fully responsive (Mobile, Tablet, Desktop)

### 🎨 Modern UI

- Dark theme with glassmorphism
- Smooth animations (Framer Motion)
